Problem 1: Remove Duplicates from Sorted Array

Problem Statement: Given a sorted array, remove duplicates in-place such that each element appears only once. Return the length of the modified array. 
•	Example: [0,0,1,1,1,2,2,3,3,4] → [0,1,2,3,4] (length = 5) 


Strategy (Slow-Fast Pointer):
•	Pointer i: Tracks the position of the last unique element found. 
•	Pointer j: Scans through the array looking for new unique elements. 
•	Action: When nums[i] != nums[j], move i forward and copy nums[j] to position i. 


Pattern
•	Two pointers (slow-fast)
•	Exploiting sorted property


Core Thinking
•	Since array is sorted, duplicates are adjacent
•	We don’t need to search globally, only compare neighbors



int removingDuplicates(vector<int>& nums, int length) {
    int i = 0;
    for(int j = i + 1; j < length; j++) {
        if(nums[i] != nums[j]) {
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1;
}
