Problem 2: Remove Element


Problem Statement: Remove all occurrences of a target value in-place. Order does not matter. 
•	Example: nums=[0,0,1,2,2,3,0,4,2,7], target=2 → k=7


Strategy (Partitioning):
•	Since order doesn't matter, use a swap-based approach. 
•	Pointer i: Scans from the left. 
•	Pointer length: Marks the boundary of valid elements from the right. 
•	Action: If nums[i] == target, swap it with nums[length] and decrement length. Do not move i yet, as the swapped element needs checking.



int removeElement(vector<int>& nums, int target) {
    int i = 0;
    int length = nums.size() - 1;
    while(i <= length) {
        if(nums[i] == target) {
            swap(nums[i], nums[length]);
            length--;
        } else {
            i++;
        }
    }
    return i;
 }
