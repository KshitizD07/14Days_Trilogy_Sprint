Problem 3: Move Zeroes


Problem Statement: Move all zeros to the end while maintaining the relative order of non-zero elements. 
•	Example: [0,1,0,3,12] → [1,3,12,0,0]


Strategy:
•	Pointer i: Tracks the next position for a non-zero element. 
•	Pointer j: Scans for non-zero elements. 
•	Invariant: Everything before index i contains non-zero elements in original order.



void moveZeroes(vector<int>& nums) {
    int i = 0, j = 0;
    while(j < nums.size()) {
        if(nums[j] != 0) {
            if(nums[i] == 0) swap(nums[i], nums[j]);
            i++;
        }
        j++;
    }
}
