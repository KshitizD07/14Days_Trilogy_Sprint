Problem 4: Merge Sorted Arrays


Problem Statement: Merge nums2 into nums1 (which has extra space) in-place. 
•	Example: nums1=[1,2,3,0,0,0], nums2=[2,5,6] → [1,2,2,3,5,6]


Strategy (Reverse Fill):
•	Fill from the back to avoid overwriting unprocessed elements in nums1. 
•	Use three pointers: i (end of nums1 data), j (end of nums2), and k (end of nums1 capacity). 


Pattern
•	Reverse two-pointer



void merge_sorted_arrays(vector<int>& n1, int m, vector<int>& n2, int n) {
    int i = m - 1, j = n - 1, k = m + n - 1;
    while(i >= 0 && j >= 0) {
        if(n1[i] > n2[j]) n1[k--] = n1[i--];
        else n1[k--] = n2[j--];
    }
    while(j >= 0) n1[k--] = n2[j--]; // Copy remaining nums2
}
