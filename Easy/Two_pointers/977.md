Problem 5: Squares of a Sorted Array


Problem Statement: Given a sorted array (with negatives), return a sorted array of their squares. 
•	Example: [-4,-1,0,3,10] → [0,1,9,16,100]


Strategy (Opposite Ends):
•	Largest squares come from the extremes (most negative or most positive). 
•	Use i at the start and j at the end; fill the result array from back to front.



vector<int> sortedSquares(vector<int>& nums) {
    int n = nums.size();
    vector<int> result(n);
    int i = 0, j = n - 1, k = n - 1;
    while(i <= j) {
        int left = nums[i] * nums[i], right = nums[j] * nums[j];
        if(left > right) { result[k--] = left; i++; }
        else { result[k--] = right; j--; }
    }
    return result;
}
