Problem 7: 3Sum


Problem Statement: Find all unique triplets that sum to zero. 
•	Core Strategy: Sort the array, then fix one element (nums[i]) and use Two Pointers to solve the remaining 2Sum problem for the target -nums[i]. 
•	Duplicate Handling (CRITICAL): You must skip identical values for i, j, and k to ensure the solution set is unique. 


Pattern
•	Fix one + two pointers


Core Thinking
•	Reduce 3Sum → 2Sum
