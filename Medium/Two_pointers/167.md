Pattern-
    Two pointers (opposite direction)
    Works only because:
        array is sorted

At any step:
    i → smallest remaining value  
    j → largest remaining value  
Check:
    sum = nums[i] + nums[j]

Case 1: sum < target
    Need bigger sum → move i forward
Case 2: sum > target
    eed smaller sum → move j backward
Case 3: sum == target
    Return indices (1-based)


No extra space needed
Better than hashmap version:
O(n) time, O(1) space
