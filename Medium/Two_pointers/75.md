Pattern-
    Three-pointer partitioning
    Extension of two-pointer → now 3 regions
    Also called:
        Dutch National Flag Algorithm

Pointer Roles-
    low  → next position for 0  
    mid  → current element being evaluated  
    high → next position for 2  

Case 1: nums[mid] == 0
    → swap with low
    → low++
    → mid++

Case 2: nums[mid] == 1
    → just move mid

Case 3: nums[mid] == 2
    → swap with high
    → high--
    → DO NOT move mid

Limited values → partitioning problem-
    If values are from a small fixed set → partition, don’t sort

Trigger conditions:
    1. Array has limited distinct values
    2. In-place required
    3. One-pass constraint
    4. No need for full sorting
